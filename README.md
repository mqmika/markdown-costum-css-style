# Markdown to HTML Converter

Bu proje, bilgisayarınızdaki `.md` (Markdown) uzantılı dosyaları kolayca görsel efektli `.html` web sayfalarına dönüştürmeniz için geliştirilmiş bir masaüstü uygulamasıdır. Kodlama dünyasında sıkça kullanılan Markdown formatındaki notlarınızı veya yazılarınızı, göze hitap eden modern web sayfaları haline getirmeyi amaçlar.

## 🚀 Özellikler

* **Karanlık Arayüz:** Gözü yormayan, minimalist saf siyah bir tema tasarımı.
* **Dinamik Dosya Takibi:** Seçtiğiniz Markdown dosyasının adı arayüzde anlık olarak güncellenir ve görünür.
* **Özel Görsel Efektler:** Dönüştürülen HTML dosyasının arkasına gömülmek üzere Kar Yağışı, Cyberpunk, Matrix veya Uzay Yıldızları gibi eğlenceli temalar seçebilirsiniz.
* **Doğrudan İndirme:** Dönüştürülen dosyayı kaydederken sistem otomatik olarak "İndirilenler" klasörünü hedef gösterir, böylece dosyanın nereye gittiğini aramakla uğraşmazsınız.

## 🛠️ Kullanılan Teknolojiler

Projenin geliştirilmesinde aşağıdaki kütüphaneler ve teknolojiler kullanılmıştır:

* **Python:** Uygulamanın ana programlama dili.
* **CustomTkinter:** Standart Tkinter kütüphanesinin üzerine kurulan, modern ve karanlık mod destekli arayüz (GUI) kütüphanesi.
* **Markdown (Python Kütüphanesi):** `.md` içeriğini standart HTML etiketlerine dönüştüren ana motor.
* **HTML5 / CSS3 / JavaScript:** Dönüştürülen çıktı dosyasındaki görsel efektlerin, canvas animasyonlarının ve sayfa düzeninin çalışmasını sağlayan web teknolojileri.

## 💻 Nasıl Çalıştırılır?

Gerekli kütüphaneleri yükledikten sonra projeyi terminal üzerinden çalıştırabilirsiniz:
